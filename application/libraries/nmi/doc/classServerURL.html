Detailed Description

A URL and associated attributes that can be used for processing CardEaseXML requests.

At least one of these needs to be supplied to the Client for processing to take place.

Parameters

◆ setTimeout()

ServerURL::setTimeout (   $timeout)

Sets the server URL read timeout in milliseconds.

If zero is specified an infinite timeout is used. For most requests a timeout of 45 seconds (45000) is recommended. A timeout of less than 30 seconds is not permitted as some authorisations may take this long.

Parameters
timeoutThe server URL read timeout in milliseconds.
